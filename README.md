# Shoeting-Frontend

# ReactNative
1. Node.js 설치
2. npm 설치
3. yarn 설치
~~~
npm install -g yarn
~~~
4. JDK 설치

# EXPO 설치
1. Expo 명령어 도구 설치
~~~
npm install -g expo-cli
~~~
2. Expo 계정 세팅
~~~
expo login --username "name"
~~~
3. Expo 앱 생성하기
~~~
expo init gproject
~~~
4. Expo 실행하기
~~~
expo start --tunnel
~~~
